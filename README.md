# Stellantis Performance Academy

This project was cloned from https://github.com/lukeplummer98/stellantis-performance-academy.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```

## Project Structure
- `src/` - Source code
- `public/` - Static assets
- `index.html` - Main HTML file
- `vite.config.js` - Vite configuration

## Notes
- This project uses Vite as the build tool.
